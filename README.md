# Benefits – הטבות עצמאים

אפליקציית React להטבות לעצמאים: בית, עסקים קרובים, קהילה, מימוש הטבות עם QR ודירוג.

## הרצה

```bash
npm install
npm run dev
```

נפתח בדפדפן: http://localhost:5173

## בנייה

```bash
npm run build
npm run preview
```

## טכנולוגיות

- React 18, Vite, Tailwind CSS, lucide-react
