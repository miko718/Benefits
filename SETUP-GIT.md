# חיבור ל-GitHub (פעם אחת)

כדי שכשתגיד **"העלה"** או **"עדכן"** האג'נט יוכל לדחוף ל-GitHub:

1. **צור ריפו חדש ב-GitHub** בשם `Benefits` (ללא README).
2. **חבר את הפרויקט:**
   ```bash
   cd /Users/mikoefargan/Benefits
   git remote add origin https://github.com/שם_המשתמש_שלך/Benefits.git
   git push -u origin main
   ```
3. מהרגע הזה – כל פעם שתגיד "העלה" או "עדכן", האג'נט יעשה commit ו-push אוטומטית.
